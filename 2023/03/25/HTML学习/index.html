用于定义HTML文档所要显示的内容，也被称为主体标签。浏览器中显示的所有文本、图像、音频和视频等信息都必须位于标签内，才能最终展示给用户。一个HTML文档只能含有一对标签标签必须在标签内，位于标签之后，与标签是并列关系。

是一个单标签，charset表示该html文件，以"utf-8"编码保存。

声明不是一个 HTML 标签；它是用来告知 Web 浏览器页面使用了哪种 HTML 版本。
